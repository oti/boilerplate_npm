# Boilerplate Static Web Dev v2

- [v1](tree/1.0.2/): EJS, Sass, Autoprefixer, mq-packer, uglifyjs(**non-bundle, non-compile**)
- [v2](tree/master): EJS, Sass, Autoprefixer, mq-packer, **typescript, webpack**

## Getting Started

Node v12.16.1

```bash
npm ci
npm start
```

## Build

```bash
npm run build
```

output to `dist/`.
