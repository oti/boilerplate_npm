# npm run-script Web Dev Boilerplate

## [v1](../../tree/1.0.2/)

- HTML{: EJS
- CSS: Sass, Autoprefixer, mq-packer
- JS: uglifyjs(**non-bundle, non-compile**)

## [v2](../../tree/master)

- HTML: EJS
- CSS: Sass, Autoprefixer, mq-packer
- JS: **TypesSript, Babel, webpack**

## development

```bash
ndenv install 12.16.1
ndenv rehash
```

```bash
npm ci
npm start
```

`localhost:3000` が起動します。

## Build

```bash
npm run build
```

`dist/` にビルドされます。

## Test

```bash
npm run test
```
